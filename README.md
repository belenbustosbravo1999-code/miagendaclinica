# MiAgendaClinica

Aplicación desarrollada para la gestión de pacientes y citas médicas.

## Tecnologías
- Angular
- Ionic
- TypeScript
- Python
- MySQL
- XAMPP

## Funcionalidades
- Registro de pacientes
- Agendamiento de citas
- Visualización de horarios
- Inicio de sesión
- Registro de usuarios
- Ver citas
- Editar citas
- Cancelar citas
- Conexión con MySQL
- Gestión de base de datos
