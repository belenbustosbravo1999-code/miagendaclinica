# Mi Agenda Clínica

## Descripción del Proyecto

Mi Agenda Clínica es un sistema desarrollado en Python y MySQL que permite gestionar citas médicas mediante una conexión a base de datos. El sistema permite iniciar sesión, registrar usuarios y administrar citas médicas.

Este proyecto fue desarrollado para la asignatura Ingeniería de Software (PRY3211).
